# ArchMind v0.5

ArchMind v0.5 introduces Brain-driven project generation,
multilingual idea interpretation, and stable local runtime deployment.

Key features:

## Brain v2
- architecture reasoning layer
- automatic template selection
- module reasoning (auth, db, dashboard, worker, file-upload)

## Multilingual ideas
- English, Korean, Japanese idea inputs supported
- idea normalization layer

## Templates
New templates:
- worker-api
- internal-tool
- data-tool

Improved templates:
- fastapi
- fastapi-ddd
- fullstack-ddd
- nextjs

## Local runtime
- deploy local
- running
- stop
- restart
- logs

## Telegram UX
New commands:
- /preview
- /inspect
- /suggest

Improved:
- /projects
- /use
- /help

## Stability
- worker-api sqlite initialization fixed
- generation pipeline reliability improvements
