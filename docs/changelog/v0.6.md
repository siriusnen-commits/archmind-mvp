# ArchMind v0.6

ArchMind v0.6 introduces a complete AI-assisted development loop.

## New features
- `/design` for architecture design documents from ideas
- `/plan` for development roadmap generation
- `/apply_plan` for automatic plan step execution
- `/next` for project-aware next-step recommendations
- richer `/inspect` project overview output
- explicit project evolution commands:
  - `/add_entity`
  - `/add_field`
  - `/add_api`
  - `/add_page`

## Improvements
- smarter domain suggestion rules
- QA/device/test-run oriented suggestions
- better `/apply_suggestion` UX guidance when no active project
- improved project overview formatting and actionability

## Development workflow
`idea`
-> `design`
-> `suggest`
-> `plan`
-> `apply_plan`
-> `inspect`
-> `next`

## Stability
- full pytest suite passing
- existing command flows preserved
