# ArchMind v0.4

Release date: 2026-03-15

## Major milestone

Brain + Local Runtime Manager

ArchMind now supports architecture reasoning before generation and can manage the lifecycle of locally running projects.

## New features

- Brain v1 architecture reasoning layer
- architecture_reasoning.json output
- Brain evaluation cases and pipeline alignment tests
- deploy target abstraction (railway / local)
- local auto deploy pipeline with /idea_local
- local runtime lifecycle commands
  - /running
  - /logs
  - /restart
  - /stop
  - /delete_project
- improved /help command UX
- runtime-neutral project templates for local and railway targets

## Current workflow

idea  
→ architecture reasoning  
→ project generation  
→ local run or deploy target selection  
→ smoke verification  
→ lifecycle operations  

## Notes

This release shifts ArchMind from a project generator/deployer toward a more complete AI-driven development environment prototype.
